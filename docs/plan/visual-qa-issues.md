대상: https://ai.joonwon.dev

뷰포트: Mobile 375px / Tablet 768px / Desktop 1280px

## 🟡 Warning — 우선 수정

### W-1. 다크모드 링크 텍스트 색상 대비 미달
- **페이지**: 전체 (로그인, 회원가입 하단 링크)
- **증상**: "계정이 없으신가요?" 일반 텍스트의 색상(`rgb(74,86,106)`)이 카드 배경(`rgba(14,17,28,0.9)`) 대비 **2.54:1** — WCAG AA 기준 4.5:1 미달
- **수정**: 다크모드에서 해당 텍스트 색상을 `rgb(150,165,190)` 이상으로 밝혀 대비 4.5:1 확보

### W-2. 모바일 카드 좌우 패딩 없음
- **페이지**: `/`, `/signup`
- **뷰포트**: Mobile 375px
- **증상**: `.card` 요소가 뷰포트 전체 너비를 채워 좌우 여백 0px — 콘텐츠가 화면 엣지에 밀착
- **수정**: 모바일에서 카드에 `padding-left/right: 16px` 또는 Tailwind `px-4` 적용

### W-3. 모바일 터치 타겟 미달 — 테마 전환 버튼
- **페이지**: 전체
- **뷰포트**: Mobile 375px
- **증상**: 테마 전환 버튼 높이 32px (권장 최소 44px)
- **수정**: 버튼 `min-height: 44px` 또는 `padding-y` 확대. `aria-label="테마 전환"` 이미 있음

### W-4. 모바일 터치 타겟 미달 — 페이지 이동 링크
- **페이지**: `/`, `/signup`
- **뷰포트**: Mobile 375px
- **증상**: "계정 만들기" 링크 55x14px, "로그인" 링크 31x14px — 높이 14px은 권장 44px의 32% 수준
- **수정**: 링크를 `inline-block`으로 변경하고 `padding: 15px 8px` 추가하거나, 부모 요소 전체를 클릭 가능하게 변경

### W-5. 입력 필드 테두리 가시성 부족
- **페이지**: 전체
- **증상**: 다크모드 테두리 `rgba(255,255,255,0.07)`, 라이트모드 `rgba(0,0,0,0.08)` — 불투명도가 낮아 입력 필드 경계가 거의 보이지 않음
- **수정**: 테두리 불투명도를 다크모드 `0.20` 이상, 라이트모드 `0.15` 이상으로 조정

### W-6. 로그인 페이지 로드 시 불필요한 401 API 요청
- **페이지**: `/` (로그인 페이지)
- **증상**: 로그인 전 상태임에도 인증이 필요한 API 엔드포인트에 요청이 발생해 401 응답 수신
- **수정**: 해당 API 호출을 인증 완료 후에만 실행하도록 조건 추가 (로그인 페이지에서는 호출 생략)

## 🟢 Minor — 개선 권장

### M-1. 레이블/서브타이틀 글꼴 크기 작음
- **증상**: 폼 레이블 11px, 서브타이틀 13px — 고령자/저시력자 접근성 저하
- **수정**: 레이블 최소 12px, 서브타이틀 최소 14px으로 상향 권장

### M-2. 입력 필드 `aria-label` 누락
- **증상**: `<input type="email">`, `<input type="password">`에 `aria-label` 없음. `<label>` 연결은 있으나 보강 권장
- **수정**: 각 input에 `aria-label="이메일"`, `aria-label="비밀번호"` 추가

### M-3. 데스크탑 폼 카드 수직 위치 오프셋
- **뷰포트**: Desktop 1280px
- **증상**: 폼 카드가 진정한 수직 중앙이 아닌 위쪽에 오프셋 위치 (상단 여백 약 178px)
- **수정**: 레이아웃 컨테이너에 `align-items: center` + `min-height: 100vh` 확인

### M-4. 미구현 경로에서 nginx 기본 404 페이지 노출
- **증상**: `/agents`, `/dashboard`, `/register` 접근 시 nginx 기본 404 페이지 노출
- **수정**: nginx 설정에 커스텀 404 페이지 추가 또는 SPA fallback으로 `/login` 리다이렉트

### M-5. Cloudflare RUM 스크립트 로드 실패
- **증상**: `https://ai.joonwon.dev/cdn-cgi/rum?` 요청이 `ERR_ABORTED`
- **수정**: Cloudflare 대시보드에서 RUM 설정 확인. 기능 영향은 없으나 분석 데이터 누락

### M-6. `.glow` 요소 기술적 overflow (모바일)
- **뷰포트**: Mobile 375px
- **증상**: `.glow` 요소 right 값이 428px로 뷰포트(375px)를 53px 초과. `body { overflow: hidden }`으로 시각적으로는 차단되나 기술적 overflow 발생
- **수정**: `.glow` 요소에 `max-width: 100vw` 또는 `right: 0` 제한 추가
